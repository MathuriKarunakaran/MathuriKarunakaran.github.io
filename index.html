PowerBI Data Professional Survey Breakdown

In this project, I created a comprehensive Power BI dashboard to showcase the findings from a survey conducted of data professionals. The dashboard provides insightful visualisations, including comparisons of Salary by Job Title, Favourite Programming Language, and Job Satisfaction. To optimise the reporting process, I employed advanced data transformation techniques in Power Query, utilising functions like split column delimiter and creating custom columns to calculate average salary, thereby enhancing the overall data structure. The resulting dashboard not only presented key survey outcomes but also served as a valuable tool for informed and data-driven decision-making.
